# Contributing

## Branch naming

- `main`: stable/demo
- `develop`: integration
- `feature/<short-name>`: nhánh làm việc

## Pull Request rules

- Không push trực tiếp lên `main` / `develop`
- 1 PR = 1 mục tiêu rõ ràng
- PR phải có: mô tả + cách test

## Definition of Done (tuần)

- Code chạy được local
- Không commit secret / .env
- Có cập nhật README/docs nếu thay đổi cách chạy
